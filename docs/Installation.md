# Laravel Grid
This package will help design data grids for your application. Inspired by the Yii2 grids widget.

# Features
+ Data Rendering
+ Per column Filters
+ Search
+ Sort
+ Row and column Buttons
+ Data Export to PDF, excel and csv (uses https://github.com/Maatwebsite/Laravel-Excel)
+ Pjax ready
